Технологии для веб-разработки

  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets) или их модификации SCSS, SASS, Post CSS, Less
  • JavaScript (или его аналог TypeScript) и связанные с ним фреймворки — React.js, Angular и Vue.js)
  • PHP, Java, Python, Go, C#, Node.js и даже Ruby
  • npm, yarm (менеджеры пакетов), git (системы контроля версий) и многие другие

История появления технологий веб-разработки

  1. 1991 - HTML
  2. 1995 - JavaScript
  3. 1996 - CSS
  4. 2005 - GIT
  5. 2009 - Node.js
  6. 2012 - TypeScript
  7. 2013 - React
  8. 2014 - Vue.js
  9. 2016 - Angular
