HTML to define the content of web pages

Hyper Text Markup Language (HTML) is the standard markup language for creating Web pages. HTML elements tell the browser how to display the content. They mark pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

HTML5 Semantic Tags: article, aside, details, figcaption, figure, footer, header, main, mark, nav, section, summary, time.

CSS to define the content of web pages

Cascading Style Sheets (CSS) is used to format the layout of a webpage. CSS can be added to HTML documents in 3 ways:

  1. Inline - by using the style attribute inside HTML elements.
  2. Internal - by using a "style" element in the "head" section.
  3. External - by using a "link" element to link to an external CSS file.

JavaScript to define the content of web pages

It can update and change both HTML and CSS. It can calculate, manipulate and validate data.
