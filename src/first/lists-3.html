Технологии для веб-разработки

  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets) или их модификации SCSS, SASS, Post CSS, Less
  • JavaScript (или его аналог TypeScript) и связанные с ним фреймворки — React.js, Angular и Vue.js)
  • PHP, Java, Python, Go, C#, Node.js и даже Ruby
  • npm, yarm (менеджеры пакетов), git (системы контроля версий) и многие другие

Programming language of the Web

  1. HTML to define the content of web pages
    1. Hyper Text Markup Language (HTML) is the standard markup language for creating Web pages.
    2. HTML elements tell the browser how to display the content. They mark pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
    3. HTML5 Semantic Tags:
      • article
      • aside
      • details
      • figcaption
      • figure
      • footer
      • header
      • main
      • mark
      • nav
      • section
      • summary
      • time
  2. CSS to specify the layout of web pages
    1. Cascading Style Sheets (CSS) is used to format the layout of a webpage
    2. CSS can be added to HTML documents in 3 ways:
      1. Inline - by using the style attribute inside HTML elements
      2. Internal - by using a "style" element in the "head" section
      3. External - by using a "link" element to link to an external CSS file
  3. JavaScript to program the behavior of web pages
    • It can update and change both HTML and CSS
    • It can calculate, manipulate and validate data
